# Nemotron 3 Super Meetup Contest - Hyderabad

Welcome to the official submission repository for the **Nemotron 3 Super Meetup Contest**, hosted by the **HydPy Community**. This contest challenges developers to build impactful community solutions using the **Nemotron-3-Super** model, leveraging its strengths in long-context reasoning and agentic workflows.

## 📅 Important Timelines

* **Submission Deadline:** 1st June 2026.



## 🚀 Contest Tracks

Participants must choose one of the following four tracks:

1. **Community Benefits Navigator:** Aid users in applying for public/nonprofit services.


2. **Open-Source Maintainer Copilot:** A GitHub helper for small community projects.


3. **Local Governance Explainer:** Simplify city council minutes and policy updates for residents.


4. **Crisis Information Assistant:** Route verified resources during regional emergencies.



## 🛠 Submission Requirements

To be eligible for prizes, your submission must include:

* **GitHub Repository:** Hosted code within the project scope.


* **Architecture Diagram:** A clear visual of your system design included in your folder.


* **YouTube Demo:** A recorded video demonstrating the project.


* **Blog Post:** A write-up shared on social media tagging **#NVIDIA #Nemotron #HydPy**.



## 📝 How to Submit

1. **Fork** this repository.
2. Create a folder under `submissions/[your_track]/[team_name]`.
3. Include your code, architecture diagram, and a `README.md` containing:
* Team Name.
* Project description.
* Link to YouTube Demo.
* Link to Medium Blog Post.
* Link to LinkedIn Post.


4. Submit a **Pull Request** to this main repository.  
5. Fill the submission form to provide your project details **https://forms.gle/E3TP4pUdMSaBfzKr5**  


## ⚖️ Judging Criteria

All projects will be scored across 6 dimensions:

* **Community Value:** Solving a real public problem.


* **Grounded Accuracy:** Faithfulness to source documents.


* **Actionability:** Immediate usefulness for the user.


* **Safety:** Avoiding harmful or "confidently wrong" advice.


* **Equity/Accessibility:** Usability for non-experts.


* **Operational Efficiency:** Latency, throughput, and cost.



## 🎁 Prizes

* **Top 3 Winners:** Amazon vouchers and NVIDIA swags.


* **2 Runners up:** NVIDIA Swags.
